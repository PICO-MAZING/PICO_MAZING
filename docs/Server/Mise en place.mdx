### Installation de RaspAP
- [Installation de RaspAP](https://raspap.com/docs/quick-start/)

```bash
curl -sL https://install.raspap.com | bash -s -- --help
```

### Installation de Docker
- [Installation de Docker](https://docs.docker.com/engine/install/debian/)

```bash
sudo apt-get update
sudo apt-get install \
    apt-transport-https \
    ca-certificates \
    curl \
    gnupg \
    lsb-release

curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

echo \
  "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/debian \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

sudo apt-get update
sudo apt-get install docker-ce docker-ce-cli containerd.io
```

### Installation de Portainer
- [Installation de Portainer](https://www.portainer.io/installation/)

```bash
docker volume create portainer_data
docker run -d -p 9000:9000 --name=portainer --restart=always -v /var/run/docker.sock:/var/run/docker.sock -v portainer_data:/data portainer/portainer-ce
```

## Installation de Micro-ROS Agent
- [Installation de Micro-ROS Agent](https://micro.ros.org/docs/tutorials/core/first_application_rtos/)

```bash
docker run -it -d -p 8888:8888 microros/micro-ros-agent:foxy udp4 --port 8888
```
