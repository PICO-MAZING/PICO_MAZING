## Lancement

```py
python main.py [-h] [-d DIM] [-a] [-nbr NOMBRE_R] [-l LOAD] [-w WRITE]
```

#### Argument -d / --dim
Type : int  
Description : Spécifie la dimension du labyrinthe. Doit être un entier supérieur à 1.  
Exemple : python script.py -d 10  

#### Argument -a / --affiche
Type : bool  
Description : Active l'affichage de la génération du labyrinthe.  
Exemple : python script.py -a  

#### Argument -nbr / --nombreR
Type : int  
Description : Spécifie le nombre de robots à utiliser. Doit être un entier égal ou supérieur à 1.  
Exemple : python script.py -nbr 2  

#### Argument -l / --load
Type : str  
Description : Charge un labyrinthe existant à partir d'un chemin spécifié. Les autres arguments ne sont pas pris en compte.  
Exemple : python script.py -l path/to/labyrinth  

#### Argument -w / --write
Type : str  
Description : Enregistre le labyrinthe généré dans un fichier à partir d'un chemin spécifié.  
Exemple : python script.py -w path/to/save/labyrinth
